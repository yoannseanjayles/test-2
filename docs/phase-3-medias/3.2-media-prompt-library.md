## 0. Blocs communs (à préfixer/suffixer à chaque prompt)

### `[DA]` — Bloc direction artistique (préfixe universel photo)

```
Premium pet accessories brand photography. Warm, quiet-luxury aesthetic:
natural oak wood, washed linen, soft wool textures. Color palette: warm
cream (#F5F1EA), greige, caramel leather, muted sage green, soft terracotta
accents. Soft golden natural window light, gentle shadows. Photorealistic,
high-end editorial quality (Kinfolk / Cereal magazine style), shot on
medium-format camera, 80mm lens look, subtle shallow depth of field.
Dignified, calm animals — never costumed or anthropomorphized.
```

### `[NEG]` — Negative prompt universel

```
--no text, watermark, logo, deformed anatomy, extra limbs, distorted eyes,
oversaturated colors, harsh flash, cluttered background, plastic-looking fur,
cartoon style, human hands with wrong fingers, glossy AI look
```

*(Modèles sans champ négatif : ajouter en fin de prompt « Avoid: text, watermarks, deformed anatomy, oversaturation, cartoon style. »)*

### `[PACK]` — Base packshot (remplace la fin du bloc DA pour les packshots)

```
Studio product photography on seamless warm greige background (#EDE8E0),
soft diffused lighting from upper left, gentle contact shadow, product
perfectly sharp and centered, no props unless specified, true-to-life
colors, catalog consistency.
```

## Règles d'exécution

1. **Cohérence de série** : générer les visuels d'une même série (cartes univers, vignettes catégories, icônes) dans la même session avec le même seed/style reference quand le modèle le permet (`--sref` Midjourney, image de référence GPT/Flux).
2. **Zéro texte généré** : tout texte (titres, cotes, prix) est ajouté en post-production (D-042).
3. **Nommage des fichiers** : `{ID}-{variante}.{ext}` (ex. `M-HOME-01-desktop.avif`) — traçabilité inventaire → production → intégration.
4. **Upscale/export** : générer au ratio cible, exporter aux tailles sources de 3.1, convertir AVIF/WebP à l'intégration (Phase 5).
5. **Contrôle qualité** : chaque visuel animalier est vérifié (anatomie, proportions du produit porté) avant intégration ; rejet systématique si l'animal paraît costumé ou caricatural (D-008).
