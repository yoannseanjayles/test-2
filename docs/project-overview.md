# Project Overview

## Vision du projet

E-commerce **premium** spécialisé dans les **accessoires pour animaux** (chiens, chats et NAC), conçu comme un produit professionnel, moderne, évolutif et documenté, pouvant être repris par n'importe quelle équipe.

## Périmètre

- **Inclus** : catalogue d'accessoires (pas d'alimentation), parcours d'achat complet, compte client avec profil animal, contenu éditorial (guides), back-office d'administration.
- **Marché de lancement** : francophone (langue FR, devise EUR).
- **Évolutions prévues (hors lancement)** : abonnement/réassort, i18n, extension de gamme.

## Fonctionnalités principales (validées à date)

- Taxonomie primaire par animal (Chien / Chat / NAC), secondaire par usage.
- Navigation mobile-first, méga-menu desktop, recherche avec autocomplétion, fil d'Ariane.
- Checkout invité, coûts transparents, paiements express, réassurance au paiement.
- Profil animal et parcours « racheter facilement ».
- Contenu éditorial séparé du catalogue mais maillé (SEO + positionnement conseil).
