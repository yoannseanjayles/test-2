# Phase 2.0 — Best Practices Summary : Page d'accueil

> **Statut** : 🟡 En attente de validation HITL
> **Équipe** : UX Researcher, UX/UI Designer, Directeur Artistique, Copywriter, Expert SEO
> **S'appuie sur** : Vision Produit (D-008), Sitemap (D-012, D-013), User Flows 1.3.

## Rôle de la page dans notre contexte

L'accueil n'est **pas** la page la plus visitée d'un e-commerce (la majorité du trafic entre par les catégories et fiches produit via SEO/ads), mais c'est la page qui **incarne le positionnement** : c'est là que le visiteur juge en quelques secondes si la marque est premium et digne de confiance. Pour nous, elle a 3 missions : *poser le positionnement*, *router vers le bon univers animal*, *initier la personnalisation*.

## 1. Éléments indispensables (recherche)

| Élément | Enseignement clé |
|---|---|
| **Hero** | 60–100 % du viewport desktop, 50–70 % mobile. **Une seule proposition de valeur claire + un seul CTA primaire** — les CTA concurrents créent de la fatigue décisionnelle. CTA orienté action (« Découvrir la sélection ») plutôt que passif (« En savoir plus »). Valeur spécifique, pas générique (« qualité au meilleur prix » = à proscrire). |
| **Au-dessus de la ligne de flottaison** | 54 %+ des utilisateurs se concentrent sur le contenu above-the-fold, surtout sur mobile : la proposition de valeur + les entrées par animal doivent y être visibles. |
| **Entrées catégories** | L'accueil doit donner un **aperçu du catalogue** (recherche Baymard 1.0) : entrées visuelles vers les univers (pour nous : Chien / Chat / NAC), pas seulement du branding. |
| **Réassurance** | Signaux de confiance visibles sans scroll profond : livraison, retours 30 jours, paiement sécurisé, avis. |
| **Preuve sociale** | Avis clients avec contexte (animal, taille) ; les marques premium intègrent la preuve sociale *dans* le récit, pas en pavé isolé. |
| **Contenu éditorial** | La fusion contenu + commerce crée l'engagement émotionnel (70 % des consommateurs émotionnellement engagés dépensent 2× plus — Capgemini) : les guides doivent apparaître sur l'accueil. |

## 2. Erreurs fréquentes à éviter

1. **Carrousel hero auto-défilant** : 60–70 % des meilleures boutiques utilisent un hero statique ; les sliders automatiques augmentent la charge cognitive et diluent le message (« deadly » — Nielsen). → Hero statique, une seule idée.
2. **Accueil-catalogue fourre-tout** : empiler 15 sections de produits tue la hiérarchie. Une page premium respire ; chaque section a un rôle.
3. **Branding sans routage** : l'inverse existe aussi — de belles images mais aucun chemin clair vers les produits. L'accueil doit router en 1 clic vers chaque univers animal.
4. **Proposition de valeur générique** : le premium se prouve par la spécificité (matériaux, curation, conseil), pas par des superlatifs.
5. **Pop-up newsletter à l'arrivée** : friction immédiate contraire au positionnement ; si capture e-mail il y a, elle se fait en fin de page ou après engagement.

## 3. Spécificités « premium » et « pet »

- **Storytelling structuré** : le héros du récit est **le client et son animal**, le problème est la médiocrité/le choix infini des marketplaces, la marque est **le guide** (curation + conseil). Le copywriting de l'accueil suivra cette structure.
- **Direction artistique cinématique** : photos lifestyle art-directées (animaux réels dans des intérieurs soignés) plutôt que packshots sur fond blanc en hero — le packshot vit sur les listings.
- **Personnalisation sectorielle** : pour les clients connectés avec profil animal, l'accueil doit s'adapter (recommandations « pour Max » ) — différenciateur clé identifié en 1.0.
- **Ancrage émotionnel pet** : le lien humain-animal est le ressort émotionnel n°1 du secteur ; le ton reste sobre (D-008 : jamais infantilisant).

## 4. Opportunités d'amélioration vs le marché

- Les benchmarks montrent que 67 % des sites leaders ont une home/navigation « médiocre à mauvaise » (Baymard 2025) : exécuter proprement les fondamentaux est déjà différenciant.
- **Sélecteur d'univers immédiat** (Chien/Chat/NAC) en premier écran : peu de généralistes le font bien ; c'est notre taxonomie (D-001) rendue visible.
- **Bloc « conseil » incarné** : mettre en avant un guide phare avec un vrai parti pris éditorial (ex. « Bien choisir un harnais ») au lieu d'un simple carrousel de blog.

## 5. Ordre de conception des pages (proposition D-019)

Pour la suite de la Phase 2 : **Accueil → Fiche produit → Listing/catégorie → Panier → Checkout → Compte → Guides → Confiance/légal.** Justification : la fiche produit et le tunnel portent la conversion (impact business), l'accueil définit le langage visuel réutilisé partout (dépendance de composants).
