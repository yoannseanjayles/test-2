# Phase 2.1 — Spécification de page : Accueil

> **Statut** : 🟡 En attente de validation HITL
> **URL** : `/`
> **Équipe** : UX/UI Designer, Copywriter, Directeur Artistique, Expert SEO, Motion Designer
> **S'appuie sur** : recherche 2.0 Accueil (D-020, D-021), Vision (D-008), Sitemap 1.2.

---

## 1. Objectif de la page

1. **Incarner le positionnement** premium (curation, conseil, esthétique) en < 5 secondes.
2. **Router** chaque visiteur vers son univers (Chien / Chat / NAC) en 1 clic.
3. **Initier la relation** : faire découvrir le conseil (guides) et, pour les clients connectés, personnaliser via le profil animal.

**KPIs de la page** : taux de clic vers catégorie/produit ≥ 55 %, taux de rebond ≤ 45 %, LCP < 2,5 s.

## 2. Structure — sections dans l'ordre

### S1. Bandeau de réassurance (annonce)
- Barre fine au-dessus du header, texte rotatif discret (rotation au chargement, pas d'animation continue) : `Livraison offerte dès 79 €` · `Retours 30 jours` · `Paiement sécurisé`.

### S2. Header global (composant partagé)
- Voir Sitemap 1.2 : logo, nav (Chien ▾, Chat ▾, NAC ▾, Nouveautés, Guides & Conseils), recherche, compte, panier. Sticky, se compacte au scroll.

### S3. Hero (statique, D-020)
- **Visuel** : photo lifestyle art-directée pleine largeur (chien ou chat dans un intérieur chaleureux, produit porté/utilisé). 75 % viewport desktop, 60 % mobile.
- **H1** : `Des accessoires d'exception pour ceux qui comptent le plus.`
- **Sous-titre** : `Une sélection exigeante de colliers, couchages et jouets — choisis pour leur qualité, dessinés pour durer.`
- **CTA primaire unique** : `Découvrir la sélection` → scroll ancré vers S4 (ou `/nouveautes` en variante à tester).
- **Version personnalisée (client connecté avec profil animal)** : H1 devient `De belles choses pour {prénom de l'animal}.` + CTA `Voir notre sélection pour {espèce/taille}`.

### S4. Entrées univers (routage, mission n°2)
- 3 cartes visuelles grand format : **Chien** / **Chat** / **NAC**. Photo + libellé + micro-texte (`Colliers, couchages, jouets…`). Clic → page animal.
- Mobile : cartes empilées plein écran l'une sous l'autre, la première partiellement visible sous le hero (incite au scroll).

### S5. Sélection curée (« Nos indispensables »)
- 4 à 8 produits max, mis en scène (mix packshot propre + vignette lifestyle au hover).
- Composant carte produit (partagé avec les listings) : photo, nom, prix, note avis, badge éventuel (`Nouveau`).
- CTA secondaire : `Voir tous les best-sellers`.
- **Client connecté** : la sélection est remplacée par des recommandations liées au profil animal (`Pour Max`).

### S6. Bloc marque (storytelling)
- Structure narrative (recherche 2.0) : *vous et votre animal (héros) / la jungle des marketplaces (problème) / notre curation (guide)*.
- 2 colonnes : photo atelier/matière + texte court (3 phrases) + CTA tertiaire `Notre histoire` → `/notre-histoire`.
- Sous le texte : 3 piliers en icônes — `Matériaux durables` · `Sélection exigeante` · `Conseil d'experts`.

### S7. Conseil incarné (guide phare)
- 1 guide mis en avant en grand (visuel + titre + accroche), pas un carrousel de blog : ex. `Bien choisir un harnais : le guide complet`.
- 2 guides secondaires en vignettes. CTA : `Tous nos conseils` → `/guides`.

### S8. Preuve sociale
- 3 avis clients sélectionnés, avec contexte animal (`Berger australien, harnais taille L`), note globale agrégée et nombre d'avis.
- Discret et intégré (fond légèrement teinté), pas de widget tiers (H6/D-011).

### S9. Newsletter (D-021 : en pied de page, pas de pop-up)
- Promesse claire orientée valeur : `Conseils d'experts et nouveautés, une fois par mois. Pas de spam.` Champ e-mail + consentement RGPD.

## 4. Contenu & ton (Copywriter)

- Ton : chaleureux, précis, sobre (D-008). Pas de superlatifs creux, pas de jeux de mots « toutou/minou ».
- Chaque section = 1 idée, ≤ 3 phrases. Les CTA sont des verbes d'action (`Découvrir`, `Explorer`, `Lire le guide`).
- Textes définitifs à produire en Phase 5 avec les contenus réels ; les libellés ci-dessus sont des propositions directrices.

## 5. Hiérarchie des CTA

| Niveau | Style (à définir en Phase 4) | Exemplaires sur la page |
|---|---|---|
| Primaire | Bouton plein | 1 seul : hero (`Découvrir la sélection`) |
| Secondaire | Bouton contour | `Voir tous les best-sellers`, cartes univers |
| Tertiaire | Lien texte fléché | `Notre histoire`, `Tous nos conseils` |

## 6. SEO & accessibilité

- **SEO** : 1 seul H1 (hero) ; H2 par section ; title `Accessoires premium pour chiens, chats & NAC — {Marque}` ; meta description orientée curation/qualité ; données structurées `Organization` + `WebSite` (avec SearchAction) ; images lifestyle en `alt` descriptifs.
- **Accessibilité** : contraste AA sur texte au-dessus des photos (overlay), focus visibles, bandeau rotatif contrôlable (pause), toutes les cartes cliquables au clavier, ordre de tabulation = ordre visuel.
- **Performance** : hero en image responsive (`srcset`, AVIF/WebP), préchargée (LCP) ; sections sous la flottaison en lazy-loading ; aucun script tiers bloquant.

## 7. Motion (Motion Designer)

- Micro-interactions sobres : fade-up léger des sections à l'entrée du viewport (une fois, 200–300 ms), zoom photo subtil au hover des cartes, transition douce du header sticky.
- Respect de `prefers-reduced-motion` : toutes les animations désactivables.

## 8. Hypothèses

- **H13** : le hero pointe vers un ancrage interne (`#selection`) plutôt que vers une page ; variante `/nouveautes` à A/B tester post-lancement.
